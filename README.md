User Management App
===================

The User Management App allows users to log in using their email and password, granting access to a suite of features for managing user information. With a user-friendly interface, the app enables the submission of user details via a form, storing the information in a Room database. Users can navigate through different sections of the app, including Home, User Menu, and Profile, to perform various actions and view stored data.

Features
--------

-   Login: Users can log in using their email and password, providing secure access to the app's functionalities.

-   Home: Upon successful login, users are directed to the Home fragment. Here, users can submit a form by entering their username, email, and phone number. Upon successful submission, the user's details are saved in the Room database, and a toast message confirms the successful addition.

-   User Menu: The User Menu, accessible through the bottom navigation, provides users with access to their stored data. Users can view all user data stored in the Room database.

-   Profile: Within the Profile section, users can see their own profile details. The logged-in user's email ID is displayed prominently. Additionally, users have the option to sign out using the logout button, which redirects them to the login page.

Installation and Setup
----------------------

1.  Clone this repository to your local machine.

    `git clone https://github.com/jayesh9999/user-management-app.git`

2.  Open the project using your preferred Android development environment.

3.  Build and run the app on an emulator or a physical device.
